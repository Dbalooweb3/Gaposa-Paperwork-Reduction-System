# Gaposa Paperwork Reduction System

A secure, modern web application designed for students and administrators at GAPOSA to significantly reduce the paperwork burden and streamline school administration.

## 🚀 Key Features

### 👨‍🎓 Student Portal
- **IT Placement Letter**: Instant generation of official letters with student details.
- **Document Preview**: High-fidelity A4 preview directly in the browser.
- **Direct Printing**: Optimized print controls for single-page A4 output (with letterhead).
- **Profile Management**: Update department and student details securely.

### 👩‍💼 Admin Dashboard
- **Template Builder**: Interactive drag-and-drop tool to map student data onto PDF/Image templates.
- **A4 Standardization**: Precise coordinate mapping (mm/pt) for pixel-perfect printing.
- **Security**: Robust protection against SQL Injection, XSS, and CSRF.

## 🛠 Tech Stack
- **Backend**: Node.js, Express.js, PostgreSQL
- **Frontend**: Vanilla JS, Modern CSS (Inter Typography)
- **PDF Engine**: Puppeteer (Cloud-optimized)
- **Infrastructure**: Ready for one-click deployment to **Render.com**

## ☁️ Deployment
This project is cloud-ready. Use the included `render.yaml` for a blueprint deployment on Render.
